Enum wasmtime_fuzzing::oracles::DiffEqResult

source · 
pub enum DiffEqResult<T, U> {
    Success(T, U),
    Poisoned,
    Failed,
}
Expand description

Result of comparing the result of two operations during differential execution.

Variants§

§

Success(T, U)

Both engines succeeded.

§

Poisoned

The result has reached the state where engines may have diverged and results can no longer be compared.

§

Failed

Both engines failed with the same error message, and internal state should still match between the two engines.
