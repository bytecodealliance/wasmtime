Expand description

The VM*Context for array-call host functions.

Its magic field must always be wasmtime_environ::VM_ARRAY_CALL_HOST_FUNC_MAGIC, and this is how you can determine whether a VM*Context is a VMArrayCallHostFuncContext versus a different kind of context.
